### Описание

Для регулярной активности в основной сети без включения виртуальных частных сетей.

#### Особенности

Запустить основной скрипт WebActivity.py до 14:40 или поменять для web1 время на более позднее

Через firefox нужно отдельно создать новый профиль в папке 
~/.mozilla/firefox/WebActivity или выбрать какой-то существующий, чтобы указать его в 36 строке файла WebActivity.py
